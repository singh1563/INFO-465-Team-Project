Screen Summaries:
- Home Screen: Provides navigation to course search, student registration, instructor schedules, and session enrollment. It also includes a quick course lookup field and overview of typical user tasks.
- Course Search Screen: Allows users to search for courses by department, instructor, or course number and view course details.
- Student Registration Screen: Allows students to log in, view enrolled sessions, and register for new sessions.
- Instructor Schedule Screen: Allows instructors to log in and view assigned course sessions.
- Session Enrollment Screen: Allows authorized users to view students enrolled in a specific session.

HTML/CSS/JS Techniques Used:
- Semantic HTML structure with header, nav, main, section, article, and footer elements.
- Shared CSS file for consistent layout, colors, font usage, cards, buttons, and navigation.
- Responsive layout using CSS grid and media queries.
- JavaScript form validation on the Home Screen quick search field.
- Relative hyperlinks between all .html pages.

Design Challenges or Decisions:
- The team used consistent file names and shared navigation so each screen can link to the others even while different members work on separate pages.
- Placeholder pages were created early to prevent broken links while teammates complete their assigned screens.
- The design uses a simple card-based layout to make the main registration functions easy to find from the Home Screen.
